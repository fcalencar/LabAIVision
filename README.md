# LabAIVision
Detecção facial do serviço Azure AI Face.

## Detectação de rostos no Vision Studio
1. No Vision Studio em https://portal.vision.cognitive.azure.com
2. Na página inicial **Get started with Vision**, selecione a guia **Face** e, em seguida, selecione o bloco **Detect faces in an image**.
